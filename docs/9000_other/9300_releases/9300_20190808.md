## ui-core-dapps - Version 3.7.0
### Features
- update versions of `lodash`, `typescript`
- `dashboard.vue`
  - add new dashboard cards
- `terms-of-use`
  - add updated terms of use
- `mailbox.vue`
  - add new entry for accepting verifications

- `mailbox.vue`
  - add missing translations
